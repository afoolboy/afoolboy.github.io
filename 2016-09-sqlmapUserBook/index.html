sqlmap 用户手册

介绍

侦查并利用sql注入

当我们添加一个bool值为真的sql表达式时（比如 id = 1 and 1 = 1 )会返回和原请求返回相同的网页，是因为后端的数据库已经解析了被注入的sql语法。前面的例子简单的描述了布尔盲注，然而，sqlmap能够尝试进行各种sql注入并且适应网站的工作流程。
sqlmap可以自动化的识别和利用各种漏洞，把url交给sqlmap，它会

  • 识别有漏洞的参数
  • 判断sql注入类型
  • 爬取后端数据库指纹
  • 根据用户设置，广泛的爬取指纹，穷举数据甚至接管数据库

直接连接到数据库

sqlmap能够监测利用五种不同的sql注入类型

  • 布尔盲注
  • 基于时间延迟的盲注
  • 报错注入
  • union联合注入
  • 多语句注入(Stacked queries)

功能

普通功能

  • 支持MySQL,Oracle,PostgreSQL,Microsoft SQL Server,Microsoft Access,IBM DB2,SQLite,Firebird,Sybase,SQP MaxDB 和 HSQLDB
  • 支持五种注入技术 boolean-based blind, time-based blind, error-based, UNION query and stacked queries.
  • 支持直接连接数据库
  • 支持提供由Burp或者WebScarab代理的日志文件，也支持从从txt文件中读取http请求包或者也支持使用Google dork而提供的一个目标的列表，同时你可以指定一个正则表达式来过滤url
  • 可以指定GET,POST,Cookie,User-Agent header,HTTP Referer注入，也可以通过逗号分隔符来指定一系列的测试参数。
  • 指定当前最大的线程数来限制盲注的速度。它也可以指定每个http请求的时间间隔。
  • 支持指定HTTP Cookie
  • 自动化处理HTTP Set-Cookie 行为，更新session并重新测试，不过你也可以指定忽略Set-Cookie行为。
  • 支持HTTP Basic, Digest, NTLM and Certificate authentications
  • 支持https
  • 支持重定义HTTP Referer header , HTTP User-Agent header
  • 可以指定输出信息的详细等级（在sqlmap中有7个等级）
  • 支持解析HTML form表单参数，伪造请求并测试表单参数是否存在注入。
  • 程序代码有着很好的健壮性的灵活性。
  • 自动化实时保存参数，当继续测试注入的时候可以直接从sesion文件中读取。
  • 支持从特定的配置文件中读取参数而不是每次都在命令行中指定
  • 支持复制后端数据库的表结构到本地的SQLite3数据库中
  • 支持解析https请求和展示各种DBMS错误信息给用户
  • 整合了其他的开源项目，比如Metasploit和w3af

指纹识别

  • 可以根据 报错信息等多种方法来判断数据库版本
  • 基础的web服务器软件和web应用指纹扫描
  • 支持取回DBMS的banner, session user and current database information.也可以检查当前用户权限是否是DBA
  • 支持列举用户，密码哈希，权限，角色，数据库、表、字段
  • 使用内置字典自动化爆破密码哈希
  • 支持暴力爆破表名和字段名
  • 支持完全的数据表dump
  • 支持自动化dump所有数据库
  • 可以寻找特定的数据库，特定的表，特定的字段，从所有的数据库中
  • 支持执行sql shell运行自定义的sql语句

接管功能

  • 支持注入用户定义的函数（MySQL PostgreSQL ）
  • 支持文件下载和上传（MySQL PostgreSQL MSSQL ）
  • 支持执行自定义命令并取回结果
  • 支持用户用户自定义函数并执行（MySQL PostgreSQL ）
  • 支持 xp_cmdshell() ，也支持存储过程的重新启用（MSSQL）
  • 支持建立一个外连的在数据库服务器和攻击机之间TCP连接
  • 通过sys_bineval()可以执行数据库内存中的msf shellcode（MySQL and PostgreSQL.）
  • 上传并执行msf的payload，通过sys_exec() on MySQL and PostgreSQL or via xp_cmdshell() on Microsoft SQL Server.
  • 支持使用特定nday执行shellcode，比如ms08068,ms09-004,ms10-015
  • 支持读取windows的registry hives.
